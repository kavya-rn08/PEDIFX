## Dataset
We use the GRAZPEDWRI-DX dataset, which is preprocessed and split into train-val-test sets. Top eight classes of AO/OTA subtypes are selected from the pre-processed dataset for the classification task. This includes: 
- 23r-M-3.1
- 23r-M-2.1
- 23u-M-2.1
- 23r-E-2.1
- 23u-E-7
- 23r-E-1
- 22r-D-2.1
- 22u-D-2.1.
